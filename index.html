Apache Server Installation and Cloud Deployment Project

In this project, I successfully demonstrated my technical skills by installing and configuring an Apache server within a virtual machine and subsequently deploying it to a cloud environment. The steps involved in this process are as follows: Virtual Machine Setup: I initiated the project by installing an Ubuntu Linux virtual machine on VirtualBox/VMware. I personalized the system by setting the hostname to my first name and last ID. Network Configuration: Enabled network settings and verified the IP address using the ifconfig command to ensure connectivity. Apache Installation: Installed the Apache package and started its service, transforming the virtual machine into a functioning web server. Web Page Creation: Developed a simple web page featuring profiles of my group members. This page was saved and tested locally via localhost. Local IP Access: Ensured accessibility by opening the website through the machine's IP address, confirming that the server was operational. Cloud Deployment: Migrated the Apache server to a cloud service provider, choosing between AWS, GCP, or Azure, for its robust capabilities. Configured the instance to run Apache and made it publicly accessible. Public Access: Tested the deployment by accessing the web page through the public cloud IP address, confirming successful deployment. Throughout this project, I meticulously documented each step and captured screenshots, demonstrating my capability in both virtual and cloud environments. This project not only highlights my technical proficiency in server management and cloud computing but also showcases my ability to execute a project from start to finish with precision and attention to detail.
